CourseFlow Mini - System Overview

Trang này dùng để giải thích toàn bộ hệ thống trước khi giao task. Nhìn theo layer: web và mobile app đăng nhập Keycloak, gateway là trust boundary, service sở hữu data riêng, event/search/chat là phần mở rộng.

Keycloak Gateway Eureka Service boundaries PostgreSQL MongoDB Elasticsearch Debezium MinIO Prometheus Grafana Training scope
Auth edge Keycloak xác thực user từ web/app. Gateway verify token và phát internal JWT training.
Core domain User, RBAC, course, enrollment là xương sống cho mọi bài sau.
Learning runtime Assignment, quiz, gradebook biến course thành flow học thật.
Platform services Media-service, MinIO, Debezium, discovery-service và observability stack có sẵn để học viên dùng và quan sát.
